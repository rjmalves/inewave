Interface de Programação para o NEWAVE

Versão: 0.0.53

Data: 23/06/2022

O inewave é um pacote Python para manipulação dos arquivos de entrada e saída do programa NEWAVE, desenvolvido pelo CEPEL e utilizado para os estudos de planejamento e operação do Sistema Interligado Nacional (SIN).

O inewave oferece:

  • Meios para leitura dos arquivos de entrada e saída do NEWAVE e programas associados: NWLISTCF e NWLISTOP

  • Armazenamento e processamento de dados otimizados com o uso de NumPy

  • Facilidades para estudo e análise dos dados utilizando DataFrames de Pandas

  • Dados estruturados em modelos com o uso do paradigma de orientação a objetos (OOP)

  • Utilidades de escritas dos arquivos de entrada do NEWAVE para elaboração automatizada de estudos
