Médias SIN (MEDIAS-SIN.CSV)¶

Visão geral do modelo¶

Os valores de médias para o SIN, localizados no arquivo geralmente denominado MEDIAS-SIN.CSV, são armazenados na classe:

class MediasSIN(dados: pandas.core.frame.DataFrame)[código fonte]¶

Armazena os dados das saídas referentes às médias de diversas variáveis para o SIN.

Esta classe lida com as informações de saída fornecidas pelo NWLISTOP e reproduzidas nos MEDIAS-SIN.CSV.

  • mes_pmo: int

  • tabela: np.ndarray
