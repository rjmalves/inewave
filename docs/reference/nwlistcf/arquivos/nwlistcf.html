Cortes da Função de Custo Futuro (nwlistcf.rel)¶

Visão geral do modelo¶

Os cortes que determinam a função de custo futuro do NEWAVE, localizados no arquivo geralmente denominado nwlistcf.rel, são armazenados na classe:

class Nwlistcf(dados: inewave._utils.dadosarquivo.DadosArquivo)[código fonte]¶

Armazena os dados dos cortes construídos pelo NEWAVE existentes no arquivo nwlistcf.rel do NWLISTCF.

Esta classe armazena os cortes da FCF de cada uma das variáveis, para cada registro e REE dentro do registro.

Cada registro possui um modelo próprio, armazenando os coeficientes dos hiperplanos em uma array específica.
