Acompanhamento do modelo PAR(p) (parp.dat)¶

Visão geral do modelo¶

O relatório de acompanhamento do modelo PAR(p) do NEWAVE, localizado no arquivo geralmente denominado parp.dat, é armazenado na classe:

class PARp(dados: inewave._utils.dadosarquivo.DadosArquivo)[código fonte]¶

Armazena os dados de saída do NEWAVE referentes aos modelos e às séries sintéticas de energia geradas pelo PAR(p) e PAR(p)-A.

Esta classe lida com informações de saída do NEWAVE e cujas saídas devem ser compatíveis com as observadas através do NWLISTOP.

property anos_estudo: numpy.ndarray¶

A lista de anos do estudo associados às tabelas de coeficientes e ordens dos modelos.

Retorna

np.ndarray

property anos_historico: numpy.ndarray¶

A lista de anos do histórico associados às séries de energia.

Retorna

np.ndarray

coeficientes_desvio_ree(ree: int) → List[numpy.ndarray][código fonte]¶

Lista dos coeficientes dos modelos PAR ou PAR-A, multiplicados pelos respectivos desvios-padrão de cada mês.

Parâmetros

  • ree: int

Retorna

List[np.ndarray]

Sobre

No caso de um modelo PAR-A de ordem p, a lista possui p + 1 coeficientes, e a última posição contém o coeficiente da componente anual.

coeficientes_ree(ree: int) → List[numpy.ndarray][código fonte]¶

Lista de coeficientes dos modelos PAR ou PAR-A.

Parâmetros

  • ree: int

Retorna

List[np.ndarray]

Sobre

No caso de um modelo PAR-A de ordem p, a lista possui p + 1 coeficientes, e a última posição contém o coeficiente da componente anual.

correlacoes_espaciais_anuais(configuracao: int) → numpy.ndarray[código fonte]¶

Correlações espaciais anuais para cada combinação de REEs em cada configuração do sistema, de maneira ligeiramente diferente da encontrada no NEWAVE (ver Sobre abaixo).

Parâmetros

  • configuracao: int

Retorna

np.ndarray

Sobre

O acesso é feito com [ree1, ree2], onde ree1 e ree2 são inteiros de 0 ao número de REEs - 1, indexados da mesma maneira do NEWAVE (0 = SUDESTE, 1 = SUL…).

correlacoes_espaciais_mensais(configuracao: int, mes: int) → numpy.ndarray[código fonte]¶

Correlações espaciais mensais para cada combinação de REEs em cada configuração do sistema, da mesma maneira encontrada no arquivo parp.dat.

Parâmetros

  • mes: int

Retorna

np.ndarray

Sobre

O acesso é feito com [mes][ree1][ree2], onde ree1 e ree2 são inteiros de 0 ao número de REEs, indexados da mesma maneira do NEWAVE (0 = SUDESTE, 1 = SUL…).
