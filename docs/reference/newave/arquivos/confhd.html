Configurações das Hidrelétricas (confhd.dat)¶

Visão geral do modelo¶

As configurações das hidrelétricas do NEWAVE, localizadas no arquivo geralmente denominado confhd.dat, são armazenadas na classe:

class Confhd(dados: inewave._utils.dadosarquivo.DadosArquivo)[código fonte]¶

Armazena os dados de entrada do NEWAVE referentes às configurações das usinas hidrelétricas.

Esta classe lida com informações de entrada fornecidas ao NEWAVE e que podem ser modificadas através do arquivo modif.dat.
