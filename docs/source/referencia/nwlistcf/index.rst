NWLISTCF
===========

O módulo ``inewave.nwlistcf`` contém as classes para leitura dos arquivos
de entrada e saída do pós-processador NWLISTCF. Cada classe representa um
arquivo específico utilizado para configuração e leitura dos resultados
de cortes de Benders gerados pelo modelo NEWAVE.
